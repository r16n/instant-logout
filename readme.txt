This plugin removes the "Are you sure" message when logging out.

== Description ==

This plugin removes the "Are you sure" message when logging out.

The shortcode `[instant-logout]` can be used to create a custom logout page. Visiting the page containing this shortcode will log out the user immediately.

== Installation ==

1. Upload custom-login-url folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Optionally, create a new page, insert `[instant-logout]` and save it.

== Frequently Asked Questions ==

= Will this plugin rewrite the logout URL? =

No, it will not.
